## Contexto específico — 4 Gráficos de Excel que Todo Analista de Costos Debería Usar

**Referencia**: `referencias/Excel/graficos-excel/GraficosExcel.jpg` — infografía vertical de Huertas Consultores con 4 tipos de gráfico de Excel, cada uno con su tabla de datos real y capturas de Excel. Se rediseña en grid 2×2 horizontal, redibujando cada gráfico de Excel como SVG propio fiel a los datos reales (no como captura de pantalla), conservando las cifras exactas de la referencia.

**Los 4 gráficos y sus datos reales**:
1. **Columnas apiladas — Costos Fijos vs. Variables por Producto**: Producto A (fijos 1000, variables 2500), Producto B (800, 2800), Producto C (1200, 1800), Producto D (950, 3100). Útil para comparar costos fijos y variables por producto o línea de negocio.
2. **Líneas — Evolución de Costos Unitarios por Mes**: Ene 4.80, Feb 5.10, Mar 4.95, Abr 5.30, May 5.00. Útil para seguir la evolución mensual de costos unitarios.
3. **Áreas — Costos Acumulados por Tipo**: Ene (Mano de Obra 3000, Materiales 4000, Energía 1000), Feb (3200, 3900, 1100), Mar (3100, 4200, 1050). Útil para ver cómo se distribuyen los costos totales acumulados.
4. **Barras horizontales — Costos por Planta**: Planta A 25,000, Planta B 21,000, Planta C 18,000, Planta D 30,000. Ideal para mostrar costos de producción por planta o centro de trabajo.

**Dirección visual**: verde real de Excel (`#217346`) como color de marca principal (uso legítimo de identidad de marca, ya que el tema es Excel específicamente), con azul marino y ámbar como series de datos secundarias, grid 2×2 con un mini-gráfico SVG real y fiel a las cifras por tarjeta (no capturas de pantalla de Excel).
