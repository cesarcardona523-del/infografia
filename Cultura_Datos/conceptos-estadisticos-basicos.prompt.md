## Contexto específico — 8 Conceptos Estadísticos Esenciales para Ciencia de Datos

**Referencia**: `referencias/Cultura_Datos/conceptos-estadisticos-basicos/ConceptosEstadisticos.jpg` — grid 4×2 de 8 conceptos, cada uno con una mini-visualización de apoyo (curva de distribución, gráfico de dispersión, boxplots, etc.). Se rediseña con el sistema propio, redibujando cada mini-visualización como SVG propio conservando su lógica visual real, evitando duplicar la paleta navy/teal ya usada en `estadisticas-ciencia-datos` y `normalizacion-ciencia-datos`.

**Los 8 conceptos (contenido real)**:
1. Distribuciones de Probabilidad — modelos matemáticos que describen cómo se distribuyen los valores de una variable aleatoria (ej. Normal, Poisson, Binomial).
2. Medidas de Tendencia Central — la media, la mediana y la moda resumen el "centro" de los datos, identificando el valor típico o más representativo.
3. Medidas de Dispersión — la varianza, la desviación estándar y el rango cuantifican la variabilidad de los datos respecto a su media.
4. Intervalos de Confianza — rangos de valores calculados a partir de una muestra que estiman, con un nivel de confianza determinado (ej. 95%), dónde se encuentra un parámetro poblacional.
5. Pruebas de Hipótesis — procedimientos estadísticos para evaluar afirmaciones sobre una población a partir de evidencia muestral; incluyen hipótesis nula (H₀), alternativa (H₁) y el cálculo del valor p.
6. Regresión Lineal y No Lineal — modelos que analizan y cuantifican la relación entre una variable dependiente y una o más variables independientes, permitiendo predicciones.
7. Análisis de Series de Tiempo — métodos para analizar datos ordenados cronológicamente, identificando tendencias, estacionalidad y patrones cíclicos.
8. Análisis de Varianza (ANOVA) — técnica que compara las medias de tres o más grupos para determinar si existen diferencias estadísticamente significativas entre ellos.
