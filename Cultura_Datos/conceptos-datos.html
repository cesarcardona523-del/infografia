Top 6 Conceptos de Datos

Seis piezas fundamentales del vocabulario de datos — y el flujo real detrás de cada una.

1
Data Warehouse

Repositorio centralizado de datos integrados, recolectados desde diversas fuentes.

Fuentes→ETL→Warehouse→Mining
2
Data Mart

Subconjunto especializado y más pequeño de un warehouse, para una línea de negocio específica.

Fuentes→ETL→Warehouse→Marts
3
Data Lake

Datos en bruto, no estructurados, en formato nativo — flexible y escalable para Big Data.

Fuentes→Lake→Transform→Listo
4
Data Pipeline

Procesos para mover datos de un sistema a otro: extracción, transformación y carga.

Origen→Staging→T&L→Destino
5
Data Quality

Grado en que los datos cumplen expectativas de exactitud, validez, completitud y consistencia.

Origen→Limpieza→Mapeo→Destino
6
Data Mining

Búsqueda y análisis de grandes volúmenes de datos para identificar patrones y extraer información útil.

Database→Análisis→Patrones→Front-End
El hilo común — warehouse, mart y lake son formas de almacenar; pipeline y quality son cómo el dato llega limpio; mining es lo que se hace una vez que ya está ahí.
