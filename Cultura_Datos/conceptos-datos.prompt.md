## Contexto específico — Top 6 Conceptos de Datos

**Referencia**: `referencias/Cultura_Datos/conceptos-datos/Conceptos.jpg` (marca "Data&Analytics", se generaliza sin su branding) — 6 conceptos fundamentales con definición y mini-flujo de proceso cada uno.

**Los 6 conceptos** (contenido real):
1. Data Warehouse — repositorio centralizado de datos integrados, recolectados desde diversas fuentes.
2. Data Mart — subconjunto especializado y más pequeño de un data warehouse, enfocado en una línea de negocio específica.
3. Data Lake — datos en bruto, no estructurados, almacenados en formato nativo — flexible y escalable para Big Data.
4. Data Pipeline — serie de procesos para mover datos de un sistema a otro: extracción, transformación y carga.
5. Data Quality — grado en que los datos cumplen expectativas de exactitud, validez, completitud y consistencia.
6. Data Mining — búsqueda y análisis de grandes volúmenes de datos para identificar patrones y extraer información útil.
