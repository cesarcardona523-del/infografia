## Contexto específico — 10 Términos Esenciales de Ciencia de Datos y Machine Learning

**Referencia**: `referencias/Cultura_Datos/terminos-ciencia-datos/Terminos.pdf` — glosario extenso de 19 conceptos de ML/ciencia de datos. Se condensa a los 10 términos más fundamentales para caber en una sola infografía, evitando redundancia con piezas ya publicadas (`normalizacion-ciencia-datos`, `estadisticas-ciencia-datos`).

**Los 10 términos seleccionados** (contenido real del PDF):
1. Dataset — instancia particular de datos usada para análisis o modelado en un momento dado.
2. Data Wrangling — proceso de convertir datos crudos en una forma ordenada, lista para análisis.
3. Outliers — puntos de datos muy distintos al resto; se detectan comúnmente con boxplots.
4. Supervised Learning — algoritmos que aprenden la relación entre variables y un target conocido.
5. Unsupervised Learning — explora la estructura de datos no etiquetados, sin variable objetivo.
6. Reinforcement Learning — un agente aprende por prueba y error usando recompensas y castigos.
7. Cross-validation — evalúa el desempeño de un modelo repitiendo el entrenamiento en k particiones distintas.
8. Bias-Variance Tradeoff — el equilibrio entre subajuste (alto sesgo) y sobreajuste (alta varianza).
9. PCA (Principal Component Analysis) — reduce la dimensionalidad conservando la mayor varianza posible.
10. Regularización (L1/L2) — penaliza coeficientes grandes en el modelo para reducir el sobreajuste.
