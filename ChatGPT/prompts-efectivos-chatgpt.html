Prompt Engineering

Cómo Escribir Prompts Efectivos para ChatGPT

Un mal prompt lleva a respuestas poco útiles; uno bien construido puede ahorrar horas de trabajo.

1 · Rol
Actúa como..."Analista de datos senior"
2 · Tarea
Crea / corrige / explica..."Optimiza esta consulta SQL"
3 · Formato
Muéstralo en..."Una tabla, un fragmento de código"
4 · Restricciones
Con estas condiciones..."Solo funciones de SQL Server 2016"
5 · Ejemplo
Basado en...Una mini-muestra de tus datos reales

Prompting Encadenado

1
Prompt inicialDame un resumen de este documento.
2
Modifica la salidaUsa ese resumen y escribe un artículo de 500 palabras.
3
Modifica el tonoCambia el tono de la respuesta para que suene más profesional.
4
Modifica el formatoConviértelo en una diapositiva por cada punto clave.

Errores Comunes

✕
Pedir algo muy general: "hazme un dashboard".
✕
No especificar la herramienta (Excel, SQL, Python...).
✕
No indicar el objetivo: comparar, filtrar, calcular.
✕
No mostrar cómo lucen tus datos reales.

Prompt
"Genera una fórmula para sumar valores solo si en otra columna dice 'Activo'."
