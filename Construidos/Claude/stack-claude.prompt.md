## Contexto específico — Stack de Claude 2026 para Líderes Empresariales

**Referencia**: `referencias/Claude/stack-claude/StackClaude.jpeg` (Henry Jiménez / Evolupedia) — 3 capas apiladas verticalmente (Claude, Claude Code, Claude Cowork), cada una con ~6-8 tareas de ejemplo y un cierre que explica a quién acelera cada capa. Se rediseña con una composición deliberadamente distinta a los grids ya usados en el repositorio: bandas diagonales apiladas tipo "cutaway", no tarjetas planas.

**Las 3 capas (contenido real)**:
1. **Claude — Socio de pensamiento** (razonamiento profundo y estrategia): fijar objetivos a largo plazo, definir la visión general, analizar la estrategia de la competencia, sintetizar datos de mercado, evaluar opciones de riesgo, ponderar trade-offs complejos, crear hojas de ruta a varios años, estrategia paso a paso. Acelera a: las personas.
2. **Claude Code — Socio de ingeniería** (código y construcción): generar código limpio, Python/JS/etc., resolver un bug complejo, analizar el stack trace, refactorizar un módulo legado, mejorar la mantenibilidad, auto-commit de cambios, integración con flujo Git. Acelera a: los equipos de ingeniería.
3. **Claude Cowork — Socio de operaciones** (automatización de workflows): automatizar tareas repetitivas, entrada de datos y programación, conectar con servicios cloud, gestionar workflows complejos, tareas entre aplicaciones, gestionar proyectos y documentos, contexto persistente. Acelera a: las unidades de negocio completas.

**Dirección visual (deliberadamente arriesgada, no plantillada)**: 3 bandas diagonales apiladas (paralelogramos, no rectángulos), fondo navy oscuro con acento clay/naranja real de marca Claude, cada banda con su propio color y ejemplos de tarea condensados en chips, franja de cierre con el "acelera a" de cada capa.
