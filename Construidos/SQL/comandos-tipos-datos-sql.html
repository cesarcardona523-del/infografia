Comandos y Tipos de Datos en SQL Server

Un mismo script real recorre las 4 familias de comandos — DML, DDL, DCL y TCL — más los tipos de datos que los sostienen.

gestion_clientes.sql
-- DDL: crear la estructura
CREATE TABLE Clientes (
  Id INT PRIMARY KEY, Nombre NVARCHAR(100),
  FechaAlta DATETIME2, Saldo DECIMAL(10,2)
);
-- DML: manipular los datos
INSERT INTO Clientes VALUES
  (1, 'Ana Ruiz', GETDATE(), 1500.00);
SELECT Nombre, Saldo FROM Clientes WHERE Saldo > 1000;
-- DCL: controlar el acceso
GRANT SELECT ON Clientes TO Analista;
-- TCL: confirmar la transacción
BEGIN TRANSACTION;
  UPDATE Clientes SET Saldo = Saldo - 200 WHERE Id = 1;
COMMIT;
