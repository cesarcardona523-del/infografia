✕ Sin Chain of Thought
Prompt
"Hazme una página web para venta de boletería"
Respuesta

✓ Con Chain of Thought
Prompt
"Piensa paso a paso: ¿qué necesita una página de venta de boletería? Primero lista los requisitos, luego diseña la estructura, y finalmente genera el código."
Respuesta
Analiza requisitos (catálogo, asientos, pasarela de pago, confirmación por email), propone arquitectura y genera código completo y funcional.
01
Descompone el problema
En vez de saltar a la respuesta, identifica sub-problemas primero.
02
Genera contexto intermedio
Cada paso que escribe se vuelve parte del input para el siguiente.
03
Reduce errores acumulados
Los pasos visibles permiten "corregir el rumbo" antes de la conclusión.

El modelo genera un token a la vez — pedirle razonar paso a paso es darle más espacio para pensar.
