10 Términos Esenciales de Ciencia de Datos y Machine Learning

Un glosario de referencia rápida — el vocabulario mínimo para entender cualquier conversación técnica sobre modelos.

Dataset

Instancia particular de datos usada para análisis o modelado en un momento dado.

Data Wrangling

Convertir datos crudos en una forma ordenada, lista para el análisis.

Outliers

Puntos muy distintos al resto del dataset; se detectan con boxplots.

Supervised Learning

Aprende la relación entre variables y un target ya conocido.

Unsupervised Learning

Explora la estructura de datos no etiquetados, sin variable objetivo.

Reinforcement Learning

Un agente aprende por prueba y error, usando recompensas y castigos.

Cross-validation

Evalúa el modelo repitiendo el entrenamiento en k particiones distintas.

Bias-Variance Tradeoff

El equilibrio entre subajuste (alto sesgo) y sobreajuste (alta varianza).

PCA

Reduce la dimensionalidad conservando la mayor varianza posible.

Regularización (L1/L2)

Penaliza coeficientes grandes para reducir el sobreajuste del modelo.

Por qué importa — este vocabulario aparece en casi cualquier discusión técnica de un proyecto de machine learning; no dominarlo dificulta evaluar si un modelo está bien construido.
