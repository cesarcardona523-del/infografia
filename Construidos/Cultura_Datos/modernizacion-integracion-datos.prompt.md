## Contexto específico — Modernizando la Integración de Datos

**Referencia**: `referencias/Cultura_Datos/modernizacion-integracion-datos/ModernizaciónDatos.pdf` — TDWI Checklist Report (Philip Russom, 2015), patrocinado por Informatica y SAS. Se generaliza sin usar marcas de los patrocinadores ni el logo de TDWI.

**Las 7 recomendaciones** (contenido real, checklist genuino):
1. Complementar la alta latencia de la integración tradicional (ETL nocturno) con una gama más amplia de técnicas de ingestión (streaming, ingestión continua).
2. Adoptar nuevas prácticas de preparación de datos ágil para exploración y analítica, sin la rigidez del modelado tradicional.
3. Integrar datos de forma que permitan acceso de autoservicio a una amplia variedad de usuarios, técnicos y de negocio.
4. Modernizar la infraestructura aprovechando nuevas plataformas de datos (Hadoop, NoSQL, bases columnares).
5. Agregar funciones de "momento oportuno" — desde lote nocturno hasta streaming en tiempo real.
6. Modernizar la funcionalidad para obtener valor de datos multi-estructurados y no tradicionales (texto, IoT, redes sociales).
7. Considerar una plataforma integrada de múltiples herramientas de gestión de datos en vez de un portafolio fragmentado.
