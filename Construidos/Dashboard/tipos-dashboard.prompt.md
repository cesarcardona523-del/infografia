## Contexto específico — Tipos de Dashboards

**Referencia**: `referencias/Dashboard/tipos-dashboard/TipoDashboard.jpg` — 4 tipos de dashboard con capturas de ejemplo (se redibujan como mini-visualizaciones propias, no se copian las capturas).

**Los 4 tipos** (contenido real):
1. Estratégicos — orientados a gerencia y alta dirección, métricas de alto nivel, decisiones a largo plazo.
2. Operacionales — monitorean procesos diarios, se actualizan en tiempo real, detectan problemas a medida que ocurren.
3. Analíticos — analizan grandes volúmenes de datos y tendencias, usados por analistas para profundizar.
4. Tácticos — seguimiento y decisiones a corto plazo, métricas de áreas específicas (marketing, satisfacción del cliente).

**Dirección visual**: grid 2×2, paleta violeta/púrpura (nueva dentro del tema Dashboard, distinta del azul de `reglas-diseno-dashboard`), cada cuadrante con ícono semántico + mini-visualización representativa del tipo de dashboard (KPI grande para estratégico, gauge en vivo para operacional, scatter/tendencia para analítico, funnel para táctico).
