Orquestación de Scripts en Python: Ejecución Programada a Nivel de Sistema Operativo

Registrar un script en el programador de tareas nativo, sin desplegar un orquestador externo para un flujo aislado.

ventas_rep.py — registro de la tarea

import sys, subprocess from pathlib import Path HORA = "08:00" SCRIPT = Path(__file__).with_name("ventas_rep.py") cmd = ( f'schtasks /Create /SC DAILY /ST {HORA} ' f'/TN "Reporte_Python" /TR \'"{sys.executable}" "{SCRIPT}"\' /F' ) subprocess.run(cmd, shell=True, check=True)
Windows: Task Scheduler vía schtasks. En Linux/macOS, el equivalente directo es una entrada de cron o un temporizador de systemd.
