Paradigmas de Aprendizaje
Supervisado — entrena con datos etiquetados.
No supervisado — encuentra patrones sin etiquetas.
Por refuerzo — prueba y error optimizando recompensas.
Auto-supervisado — genera etiquetas a partir de los propios datos.
IA Generativa
LLM — modelo entrenado con grandes volúmenes de texto.
GAN — dos redes que compiten para generar datos realistas.
RAG — combina modelos con bases de conocimiento externas.
Modelos de difusión — generan datos eliminando ruido progresivamente.
Agentes de IA
Agente autónomo — percibe, razona y actúa sin intervención constante.
Orquestación — coordina múltiples pasos o herramientas.
MCP — estándar para conectar modelos con datos y herramientas.
IA conversacional — sistemas que interactúan mediante lenguaje natural.
Comportamiento & Problemas
Alucinaciones — respuestas plausibles pero incorrectas o inventadas.
Sobreajuste — el modelo memoriza en vez de generalizar.
Sesgo algorítmico — intenta eludir restricciones del sistema.
Deriva del modelo — pérdida de rendimiento por cambios en los datos.
Evaluación & Métricas
Accuracy — proporción de predicciones correctas.
Precision / Recall — precisión de positivos / cobertura de positivos reales.
Benchmark — prueba estandarizada para comparar modelos.
F1 Score — media armónica entre precisión y recall.
Seguridad & Ética
IA explicable (XAI) — hace comprensibles las decisiones del modelo.
Guardrails — restricciones para prevenir usos indebidos.
Privacidad de datos — protección de información personal.
Red teaming — pruebas para identificar vulnerabilidades.
Por qué importa — este vocabulario compacto es la base para leer cualquier arquitectura, paper o producto de IA sin perderse en la jerga.
