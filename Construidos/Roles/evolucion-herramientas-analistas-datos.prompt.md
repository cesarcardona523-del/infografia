## Contexto específico — Evolución de las Herramientas que Debe Dominar un Analista de Datos

**Referencia**: `referencias/Roles/evolucion-herramientas-analistas-datos/HerramientasAnalistasDatos.jpeg` (Data & Analytics) — 3 columnas planas (2021 / 2026 / 2027) con ~20 herramientas cada una (60 en total). Se condensa a 7 líneas de evolución reales, trazadas con conectores curvos entre eras (composición "río de linaje", deliberadamente distinta de una tabla de 3 columnas planas) — 5 con evolución completa en las 3 eras, 2 como herramientas emergentes que aparecen recién en 2026.

**7 líneas de evolución seleccionadas (contenido real, condensado de las 60 herramientas originales)**:
1. Microsoft Excel → Excel (Power Query, Power Pivot) → Excel + Copilot
2. SQL Server → SQL (SQL Server, PostgreSQL, MySQL, Oracle) → SQL + IA Generativa
3. Power BI → Power BI → Power BI + Microsoft Fabric
4. Tableau → Tableau → Tableau Pulse (IA)
5. Dashboards → Dashboards interactivas → Analítica aumentada
6. *(nueva en 2026)* Python → Python + Agentes IA
7. *(nueva en 2026)* APIs REST → APIs + MCP (Model Context Protocol)

**Dirección visual**: 3 franjas de era (Hace 5 años / Hoy / Tendencia 2027) con 7 filas de linaje, cada una con su propio color categórico y flechas curvas tipo "río" conectando la misma herramienta a través de las eras — en vez de la tabla plana de 3 columnas del original.
